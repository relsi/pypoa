Sobre

O PyPoA é um evento que objetiva reunir a comunidade Python de Porto Alegre e região metropolitana, a fim e promover a troca de conhecimentos, networking, e apresentar conteúdos relacionados com a linguagem de programação python, sua comunidade e ferramentas, tanto para programdores da linguagem quanto para novos programadores, como alunos oriundos de cursos de TI ou profissionais de outras áreas interessados em programação. Participe!

A linguagem de programação Python é uma das mais linguagens mais utilizadas no mundo, seja para o desenvolvimento de aplicações web, ou para projeto na área científica, sendo a principal ferramenta de muitas empresas como Google, Facebook, Nasa, etc.

0 PyPoA acontecerá no dia 09/11/2019 na PUC-RS. E terá início às 09:00.

A participação no evento é gratuita e as vagas são limitadas.
